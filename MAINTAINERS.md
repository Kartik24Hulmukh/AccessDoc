# Maintainer responsibility gate

This artifact does not assign identities without consent.

Before publication, a human repository owner must record verified GitHub handles for:

- Release owner — tags, artifacts and withdrawal.
- Security owner — private reports and advisories.
- Accessibility reviewer — UI, generated-report and claims review.

Publication is blocked until all three responsibilities have accepted owners. Governance must not imply a team, foundation or response capacity that does not exist.
