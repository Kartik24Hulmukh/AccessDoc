# Rollback runbook

Triggers: failed health/core journey, security/privacy exposure, broken PDF, serious accessibility regression, missing telemetry, resource exhaustion, or artifact mismatch. Freeze changes; capture IDs; obtain approval; route traffic to the previous immutable digest; run health/version/generate/download/PDF smoke; verify alert recovery; preserve evidence; record RTO, data impact, approver, and follow-up. No data restore is required for the default ephemeral deployment.
