## Detailed findings

### accessdoc.fr — KNOWN COLLISION (HIGH RISK)

- **What it is:** A French-language AI-powered document accessibility tool for educators (transcribers, teachers adapting resources for visually impaired students)
- **Features:** OCR, image description, document restructuring, accessibility analysis
- **Status:** Active, login-protected service, © 2026 AccessDoc
- **Overlap:** **Direct overlap** — both products deal with document accessibility, though AccessDoc (ours) focuses on web accessibility evidence while accessdoc.fr focuses on document remediation for education
- **Recommendation:** Do NOT use "AccessDoc" in France without legal review. Consider alternative names or geographic restrictions.

### accessdoc.com — Parked domain

- Listed for sale on GoDaddy at $9,995
- No active service — just a parking page
- Could be purchased if the owner decides the name is worth it

### accessdoc.de — Different product

- Appears to be a customer data platform / blog site
- Different market (data analytics, not accessibility)
- Lower collision risk but could cause brand confusion

### accessdocs.com (plural) — Document management company

- ACCESSdocs Industries Inc, Elgin, Illinois
- Document imaging, forms processing, compliance management
- Different market but similar name

### USPTO TESS

The USPTO Trademark Electronic Search System (TESS) is not accessible via automated HTTP requests (it requires browser interaction with JavaScript). The owner must manually search at:

1. Go to https://tmsearch.uspto.gov/
2. Search for "accessdoc" (Word Mark search)
3. Also search "access doc" (with space)
4. Check both live and dead records
5. Review International Class 009 (software) and Class 042 (SaaS)
