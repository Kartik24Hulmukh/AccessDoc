# ADR-001 — preserve AccessDoc name for current PR

Status: temporary hold pending human review.

[VF] Adjacent names include DocAccess, ACCESSdocs and AccessibleDocs. [SI] Search and verbal confusion risk is material. Decision: do not rename inside the launch-hardening overlay. Human merges current work first; launch overlay second; any rename/refactor is a separate reviewed PR so code changes, legal review and migration remain auditable.
