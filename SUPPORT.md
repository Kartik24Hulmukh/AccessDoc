# Support

| Request | Channel |
|---|---|
| Reproducible defect | Bug issue form |
| Accessibility defect | Accessibility issue form |
| Feature proposal | Feature issue form |
| Setup question | GitHub Discussions/Q&A, if enabled |
| Vulnerability | Private process in `SECURITY.md` |

Support is community best-effort and creates no service-level agreement. The project does not provide legal/compliance advice or guarantee interpretation of client findings. Public reports must use synthetic, sanitized data.
